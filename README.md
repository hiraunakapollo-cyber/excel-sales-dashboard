# 📊 Vrinda Store Annual Sales Dashboard (Excel)

## 📌 Overview
This project is an interactive Excel dashboard built to analyze retail store sales performance for the year 2022.

## 🚀 Features
- Monthly Sales vs Orders trend analysis
- Gender-based sales insights (Men vs Women)
- Order status breakdown (Delivered, Returned, Cancelled, Refunded)
- Top 5 states by sales
- Sales channel performance (Amazon, Flipkart, Myntra, etc.)
- Interactive filters using slicers (Month, Channel, Category)

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## 📈 Key Insights
- Majority of orders were successfully delivered (~92%)
- Women contributed significantly higher sales (~64%)
- Amazon was the leading sales channel
- Maharashtra generated the highest revenue

## 📂 Files
- `Vrinda_Store_Dashboard.xlsx`

## 🎯 Purpose
To practice and demonstrate data analysis and visualization skills using Excel.
